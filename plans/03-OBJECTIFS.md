# 03 — Objectifs

## Mission

SaaS de gestion d'abonnements pour profils créateurs — sans compte traditionnel, identité = handle social, accès = code temporaire.

## Cibles 2026

| Horizon | Profils actifs | CA estimé / mois | Commission 25% |
|---------|----------------|------------------|----------------|
| Phase 1 (break-even) | 2 | ~4K EUR créateur | ~1K EUR agence |
| Phase 2 (croissance) | 3 | ~45K EUR créateur | ~11K EUR agence |
| Phase 3 (scale) | 5+ | 75K+ EUR | 18K+ EUR |

Break-even : ~2 profils actifs (charges fixes ~45 EUR/mois).

## KPIs principaux

- **DAU profils** — visiteurs uniques / jour par model_id
- **Conversion code → tier payant** — objectif 8-15%
- **ARPU fan** — revenu moyen par fan actif
- **Rétention M1 / M3** — % fans abonnés toujours actifs 1/3 mois plus tard
- **Commission agence** — 25% sur toutes transactions (packs, crédits, tips)

## Différenciation

- Pas de compte permanent (low friction)
- Identité sociale (Snap/Insta/phone handle)
- Livraison contenu native (Snap privé) + monétisation Heaven
- Multi-profil sous une infrastructure unique
- Intégration ecosystem SQWENSY (BEACON, sync OS)

## Risques prioritaires

1. Paiements — Stripe/Revolut live (KYC creator)
2. Captation de fuite fans vers Snap pur (churn)
3. Multi-profil opérationnel (migration gret → m1 déjà faite, reste YUMI/RUBY live)
4. Meta App Review Instagram Agent (permission `instagram_manage_messages`)
